## 安装步骤

+ `文件→偏好设置→外观→打开主题文件夹`

+ 下载`colour.css`到`主题文件夹`

+ 重新运行`Typora`

+ 在`主题`选择`colour`即可

  

**说明** *支持六级无序列表和有序列表*；若有序列表内容超出一行，序号会单独一行显示



**注** *`colour.min.css`可用于网页Typora主题*
